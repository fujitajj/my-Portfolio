GitHub

仕事で関数電卓を使って角度を求めることが多いため今回作ってみました。
調べると似たようなサイトは既にあったのですが、今回ロジックは自分で考えると決めていたので、あまり参考にしていません。生のJavaScriptを使用しています。
ｘ、ｙ、ｒの三辺のうち二辺を入力すると角度が求められます。クリアボタンを実装し、押すとすべての入力をリセットできます。
次回への改善点としては、角度がわかっている場合に辺を求めることもできるように出来ればよかった点と、
計算できない数値が入力された場合や、3か所にに値が入力された場合の処理も考えればよかった点です。
